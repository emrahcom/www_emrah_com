getmail veya fetchmail'den sadece bir tanesi kullanılacak.
getmail daha iyi bir seçenek

getmail
-------
	* aptitude install getmail4 (root)

	* klasor ve dosyalar
		cd
		mkdir .getmail
		chmod 700 .getmail
		cd .getmail
		touch hesap1@gmail.com
		touch hesap2@gmail.com
		touch hesap1@isp.com
		chmod 600 *

	* örnek bir ayar dosyasi ( ~/.getmail/hesap1@gmail.com )
		[retriever]
		type            = SimplePOP3SSLRetriever
		server          = pop.gmail.com
		username        = hesap1
		port            = 995
		password        = xxxx

		[destination]
		type            = MDA_external
		path            = /usr/bin/procmail

	* test
		getmail --rcfile hesap1@gmail.com
	
	* crontab -e
		*/15 * * * * getmail -q --rcfile hesap1@gmail.com
		*/15 * * * * getmail -q --rcfile hesap2@gmail.com
		*/15 * * * * getmail -q --rcfile hesap1@isp.com


		
fetchmail
--------
	* aptitude install fetchmail  (root)


	* $HOME/.fetchmailrc
	
		poll pop.gmail.com with protocol pop3,
			user hesap1 there is emrah here ssl,
			with password xxxx;
		
		poll pop.gmail.com with protocol pop3,
			user hesap2 there is emrah here ssl,
			with password xxxx;
		
		poll mail.isp.com with protocol pop3,
			user hesap1 there is emrah here,
			with password xxxx;
		
		mda '/usr/bin/procmail -f fetchmail'


	* chmod 600 .fetchmailrc


	* test

		fetchmail -vk


	* crontab -e
	
		*/15 * * * * fetchmail -s

		set mbox                = "~/mail/a_inbox/"
		set spoolfile           = "~/mail/a_inbox/"
		set folder              = "~/mail/"
		set record              = "~/mail/a_sent/"

		alternates hesap1@gmail.com hesap2@gmail.com hesap1@isp.com

		mailboxes +a_inbox
		mailboxes +a_sent
		mailboxes +b_gmail_hesap1
		mailboxes +b_gmail_hesap2
		mailboxes +b_isp_hesap1
